# IAsset
[Git Source](https://github.com/alchemix-finance/alchemix-v2-dao/blob/ede6fa522daa0fff2c20e5420d5e76d74abb70c3/src/interfaces/balancer/IAsset.sol)

*This is an empty interface used to represent either ERC20-conforming token contracts or ETH (using the zero
address sentinel value). We're just relying on the fact that `interface` can be used to declare new address-like
types.
This concept is unrelated to a Pool's Asset Managers.*
